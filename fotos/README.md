# Pasta de fotos

## Onde colocar cada imagem

- Liderança:
  - `fotos/lideranca/sophia.jpeg`
  - `fotos/lideranca/morgana.jpeg` (quando você tiver a foto)

- Membros:
  - `fotos/membros/`

## Dica de nomes
Use nomes simples, sem acento e sem espaço:
- `morgana.jpeg`
- `giulia.jpeg`

Se você mudar o nome do arquivo, lembre de atualizar o `src=""` no HTML.
